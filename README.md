# Street Fighter (Beat 'Em Up)

Street Fighter Game on Allegro C++
I built this game in 4 Weeks (part time) as a project for a course Object Oriented Programming at my University.

# Dep
```
make dep-install
```

# Run
```
make run
```

![](res/images/splash.png)


Key | Action | 
--- | --- | 
ESC | Exit | 
Space | Jump | 
Arrow Left | Left | 
Arrow Right | Right | 
Arrow Down | Crouch | 
a | Punch | 
s | Kick |
